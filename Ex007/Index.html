principais formatações

Negrito/Destaque

nesta frafre vamos colocar vamos a negrito isdo aqui como ficou? usamos <b> e </b>

essa é outra frase que vamos a negrito também o que achou usamos o <strong> e </strong>?

itálico/ênfase?

vamos destar a segunte: Fernando Domingos Teia o que achou? usamos <i> e</i>

agora vamos destar o nome de: Ester Albano o que voce achou? usamos<em> e</em>

texto marcado

podemos criar também um texto marcado usando a tag <mark> e </mark>

E no outro parágrafo temos outros texto marcado no final

Textos grande e peguenos

Estam ciando um texto grande e outro texto pequeno nesse parágrafo usando as tags:<big>-</big> e <small>-</small>

texto deletado

podemos marcar um texto como excluido para indicar que ele deve ser lido,mas não considerado usando a tag:<del>-</del>

texto insirido

podemos marcar um texto como inserido para mostrar que ele foi adicionado depois, usando a tag:<ins>-</ins>

sublinhado

existe também o sublinhado com a tag <u>- </u> não (semântico)

texto sobrescrito

para inserir coisas do tipo: ×20+3 usando a tag:<sup>-</sup>

texto subscrito

para inserir coisas do tipo:H2O usando a tag:<sub>-</sub>
